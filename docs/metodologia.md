# Metodologia

A analise sintetica usa indicadores simples para leitura executiva de uma carteira de projetos.

## Indicadores

- Progresso percentual
- Nivel de risco
- Prazo restante em dias
- Custo planejado
- Custo real
- Beneficio estimado

## Formulas

```text
desvio_custo = custo_real - custo_planejado
indice_beneficio_custo = beneficio_estimado / custo_planejado
```

## Interpretacao

O objetivo e apoiar leitura de portfolio, priorizando projetos com alto risco, baixo progresso, grande exposicao de custo ou melhor relacao beneficio-custo.
